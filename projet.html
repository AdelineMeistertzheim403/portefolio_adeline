Projet CCI Campus

De septembre 2017 à Juin 2018 j'ai fait une formation de niveau Bac+2 Analyste développeur d'applications informatique à la CCI Campus de Colmar.
Lors de cette formation nous avions 2 projets à rélaliser les voici :

1er projet : Application web pour créer des sondages

Le premier projet était une application web pour créer des sondages, elle était développer en C# ASP.NET, c'était un projet en binôme.

2ème projet une application en C++

Le 2ème projet à faire était une application en C++, nous étions libre de faire ce que nous voulions, c'était un projet individuelle, j'ai choisi de faire un tetris, pour la libraire graphique j'ai choisi la librairie C++ SFML.

Projets UHA 4.0

Projet fil rouge de 1ére année

Pour ce projet fil rouge de 1ère année il nous à été demander de choisir entre plusieurs API correspondant à différentes thèmatiques.
Parmi les thèmatiques proposées j'ai choisi la thèmatique music, j'avais donc 2 API, une contenat les informations des groupes de musiques, et l'autre contenant les informations des albums correspondant à chacun des groupes.
Le but du fil rouge était de créer une application web en utilisant les API, j'ai fait le choix de faire un site permettant de créer des playlists. Pour réaliser ce site web , j'ai utilisé pour le front-end HTLM 5, CSS 3 et Javascript, et pour le back-end PHP 8 et mySQL pour la base de données, puis dans un 2ème temps j'ai refait le front-end en utilisant le framework Angular.

Hackaton de Noël 2024

Le 20/12/2024 les formateurs de l'uha 4.0 nous ont proposès un hackaton de Noël, le but était de créer un site web permettant à un utilisateur de publier une chanson de Noël, de voter pour sa chanson préférer, à un administrateur de valider les chansons publiés.
Pour rélaliser ce site nous avions la journée, de 09h à 16h, avec mon équipes nous avions fait le choix de vne pas utiliser de framework mais de le faire en HTML, CSS , Javascript et PHP.
C'est notre équipe qui à rermporté ce hackaton.
